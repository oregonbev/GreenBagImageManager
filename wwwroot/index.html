Instructions:
  • Press '1', '2', '3', '4' (or 'c', 'p', 'g', 'b') to select the type of box to draw (can, plastic, glass, or belt)
  • Press and hold the mouse to draw a box
  • Clicking a box will select it
  • If mlitiple boxes overlap, clicking will select the next box in that spot
  • Press 'r' to remove the selected box
  • Press 'c' to clear ALL markup. A confirmation will appear
  • Press the "Save" button to save
